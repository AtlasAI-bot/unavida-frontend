# Chapter 1 Image Organization

Store section images in this structure:

- `section-1-1/`
- `section-1-2/`
- `section-1-3/`
- `section-1-4/`
- `section-1-6/`
- `section-1-7/`
- `section-1-8/`
- `section-1-10/`

## Naming convention (required)

Use this exact pattern so Atlas can auto-map images to sections:

`ch1_sX_Y_v01.png`

Examples:
- `ch1_s1_1_v01.png`
- `ch1_s1_2_v01.png`
- `ch1_s1_3_v01.png`
- `ch1_s1_4_v01.png`
- `ch1_s1_6_v01.png`
- `ch1_s1_7_v01.png`
- `ch1_s1_8_v01.png`
- `ch1_s1_10_v01.png`

If you generate alternates, increment version:
- `ch1_s1_7_v02.png`
- `ch1_s1_7_v03.png`

## Best format
- PNG preferred
- 16:9 ratio preferred
- Recommended size: 1920x1080
